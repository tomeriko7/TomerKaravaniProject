Game instructions:


The purpose of the game:
Create a sequence of three symbols (X or circle) in a row, column, or diagonally on the game board.


How to play?



The game board:
The board consists of 9 squares (3x3). At the beginning of the game all the squares are empty.


Division of roles:
The first player always plays with "X". The other player plays with "O".


The course of the game:
Each player in turn clicks on an empty square on the board. The current player's symbol (X or circle) will fill the selected slot.


Deciding the game:
The game ends in one of the following ways:
*Win: If one of the players forms a sequence of 3 identical symbols in a row, column or diagonal.
*Draw: If all slots are full and there is no winner. A message about the result of the game will appear at the end of the game.


Starting a new game:
Clicking the "Start New Game" button will reset the board and allow the game to be restarted.
The first player will always be "X" in the new games as well.
